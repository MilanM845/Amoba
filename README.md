# Amőba
Készítette: Magyar Milán (F3IHCB)

Játék Részletei
A játék két játékos között zajlik: az egyik a játékos, a másik pedig az AI.

A Játékról
Kezdés: A felhasználó kezd, és lerak egy X-et.

AI köre: Ezután az AI lerak egy O-t.

Játékos Köre: A felhasználó ismét lerak egy X-et, és a játék így megy tovább, amíg valaki eléri a győzelmet.

A győzelmeket egy adatbázisban menti és a végén a program ki is írja az eredményeket.

Győzelem: A játék addig folytatódik, amíg valaki elér 5 egymás melletti X-et vagy O-t. Az aki ezt eléri az nyer.
